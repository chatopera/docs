# 意图识别

## 概述

在应用聊天机器人的场景中，有一大类是任务驱动型的问答，比如来访者说“我想购买车票”，那么接下来机器人围绕订票这个需求，询问出行时间、目的地等信息，并将收集到的信息返回给聊天机器人开发者，完成订单确认、支付和下单订票等。这就是一个典型的任务驱动的问答，或者说识别意图并根据意图收集相关信息为目的。该种类型对话，在 Chatopera 云服务中，就是通过**意图识别**模块支持。
