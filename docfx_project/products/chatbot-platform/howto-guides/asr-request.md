# 调用语音识别服务

Chatopera 机器人平台支持语音识别服务，将语音转化为文本内容，即 ASR（Automatic Speech Recognition）。

## 支持语言

目前，Chatopera 机器人平台只支持简体中文（`zh_CN`）的语音识别，当机器人语言设置为 `zh_CN` 时可以使用。
