# 多轮对话

## 概述

多轮对话基于规则，使用*对话脚本*，并结合 JavaScript*函数*实现固定话术和固定流程的聊天服务。聊天机器人开发者使用[多轮对话设计器](/products/chatbot-platform/conversation/cde.html)实现满足企业业务需求的聊天机器人。
